# ADR-004: PDF con Haru PDF (hbhpdf)

## Estado
Aceptado

## Contexto
El proyecto .NET genera PDFs A4 con QuestPDF (fluent API, componentes). Necesitamos funcionalidad equivalente en Harbour.

## Decisión
Usar `hbhpdf` (contrib que envuelve libharu). Haru PDF es una librería C gratuita que genera PDFs sin dependencias externas. hbhpdf expone funciones para crear documentos, páginas, tablas, texto e imágenes.

## Consecuencias
Positivas:
- Sin dependencias externas
- hbhpdf incluido en Harbour contribs
- Suficiente para facturas A4 con texto, tablas y líneas

Negativas:
- API procedural (no fluida como QuestPDF)
- Sin soporte de componentes reutilizables
- La tabla hay que dibujarla manualmente (posiciones X/Y)
- Limitado en tipografía avanzada
