# ADR-002: Cross-compilación Windows desde Linux

## Estado
Aceptado

## Contexto
Desarrollamos en Linux pero debemos generar ejecutables para Windows 11 64-bit.

## Decisión
Usar MinGW-w64 como toolchain cross-compilador. Harbour soporta cross-compilación vía `HB_PLATFORM=win HB_COMPILER=mingw64` usando `x86_64-w64-mingw32-gcc`. HWGUI para Windows usa Win32 API directa, sin dependencias externas. El build produce un ejecutable nativo Windows de 64-bit.

## Consecuencias
Positivas:
- Un solo Makefile produce Linux y Windows
- Mismo código fuente Harbour para ambas plataformas

Negativas:
- HWGUI Linux usa GTK2 (diferente backend que Win32 API)
- Algunas contribs pueden necesitar ajustes por plataforma
- No podemos probar el binario Windows en Linux (Wine sí)
