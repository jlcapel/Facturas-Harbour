# ADR-007: Multi-idioma

## Estado
Aceptado

## Contexto
El proyecto .NET soporta 5 idiomas (ES, EN, CA, EU, FR) via archivos .resx con binding reactivo.

## Decisión
Usar el mismo sistema de claves que el proyecto .NET. Las traducciones se cargan desde arrays Harbour indexados por clave. Las claves se extraen literalmente de los archivos .resx del proyecto .NET en `/home/jose/programacion/Facturas/Facturas/Resources/`.

## Consecuencias
Positivas:
- Mismas claves que el proyecto .NET — misma BD de traducciones
- Fácil sincronizar traducciones entre proyectos

Negativas:
- Sin binding reactivo — hay que refrescar la UI manualmente al cambiar idioma
- Las cadenas se cargan en memoria al iniciar
